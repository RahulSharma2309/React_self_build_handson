# Concept Notes

This folder explains the thinking behind each story.

Use the matching files together:

- `docs/stories/01-blank-vite-app.md` tells you what to build.
- `docs/concepts/01-blank-vite-app.md` explains why each move matters and which React ideas you are touching.

Read the concept note first, then do the story. When a step feels confusing, come back here and read the matching section again.

## Learning Pattern

Every concept note follows the same pattern:

- The mental model.
- The important files.
- What each move teaches.
- Mistakes to watch for.
- A check-yourself section.

The goal is that you do not just finish the app. You should be able to explain why each file exists and why each library enters the project at that moment.
