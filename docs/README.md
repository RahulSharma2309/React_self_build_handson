# Build Stories

This folder is your rebuild path for the VoltShop React app.

The finished app already exists in `React Learning`. This repo starts almost empty on purpose. You will rebuild the same frontend in small stories so each React concept appears when you need it.

## How To Use These Stories

1. Pick the next story from `story-map.md`.
2. Open that story file from `docs/stories/`.
3. Type the code yourself. Do not copy full files from the finished app first.
4. Run the app after every small step.
5. When stuck, compare with the matching file in `React Learning/frontend/src`.

## Rules For Learning

- One story at a time.
- One concept at a time where possible.
- Keep the app working after every story.
- Use the finished app as a reference, not as the place you work.
